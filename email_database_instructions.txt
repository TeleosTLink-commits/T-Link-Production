# T-Link User Email Database Instructions

## Navigating to Project Root

Open PowerShell and run:

    Set-Location C:\T_Link
    # or
    cd C:\T_Link

## Connecting to the Database

You will be prompted for the password (default: Ajwa8770):

    psql -h localhost -U postgres -d tlink_db

## Common User Management Commands

### View All Users
    psql -h localhost -U postgres -d tlink_db -c "SELECT * FROM users;"

### Add a User
    psql -h localhost -U postgres -d tlink_db -c "INSERT INTO users (id, username, email, password_hash, first_name, last_name, role, is_active) VALUES (uuid_generate_v4(), 'username', 'user@example.com', 'hashed_password', 'First', 'Last', 'lab_staff', true);"

### Delete a User
    psql -h localhost -U postgres -d tlink_db -c "DELETE FROM users WHERE email = 'user@example.com';"

### Change a User's Role
    psql -h localhost -U postgres -d tlink_db -c "UPDATE users SET role = 'admin' WHERE email = 'user@example.com';"

### Deactivate a User
    psql -h localhost -U postgres -d tlink_db -c "UPDATE users SET is_active = false WHERE email = 'user@example.com';"

---

Replace values (like email, username, etc.) as needed for your use case. All commands should be run from the C:\T_Link directory in PowerShell.
